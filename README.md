# Installation

## pre requisites

* install [ollama](https://ollama.com/) and download the models "llama3.2" and "qwen2.5-coder:0.5b"
* install jack-audio for your operating system

## installation

```
git clone https://github.com/polymorphicengine/llami
cd llami
cabal run
```
